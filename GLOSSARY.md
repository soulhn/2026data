# GLOSSARY.md — UI 용어 사전 & 지표 정의

> 대시보드에 표시되는 모든 지표명·단위·표기를 통일하기 위한 기준 문서.

---

## A. UI 표기 규칙

| # | 규칙 | 올바른 예 | 잘못된 예 |
|---|------|----------|----------|
| 1 | **수식어 + 지표** 사이 공백 | `평균 모집률`, `평균 출석률` | `평균모집률`, `평균출석률` |
| 2 | **복합명사** 공백 | `기준 매출`, `실제 매출`, `비례 손실액` | `기준매출`, `실제매출` |
| 3 | **st.metric**: 레이블에 단위 생략, 값에 포함 | label=`"평균 출석률"` value=`f"{rate:.1f}%"` | `"평균 출석률(%)"` |
| 4 | **테이블 컬럼**: 괄호 단위 허용 | `만족도(점)`, `출석률(%)`, `수료율(%)` | — |
| 5 | **차트 축**: 공백+괄호 단위 | `모집률 (%)`, `매출액 (원)` | `모집률(%)` |
| 6 | **기간 표기**: 공백+괄호 | `취업률 (3개월)`, `취업률 (6개월)` | `취업률(3개월)`, `3개월취업률`, `3개월 취업률` |
| 7 | **숫자 포맷**: 천 단위 쉼표 | `1,234명`, `12,345,600원` | `1234명` |
| 8 | **한글만 사용**: UI 텍스트에 한자 혼용 금지 | `분석`, `경쟁` | 한자 혼용 금지 |

## B. 전체 지표 정의 & 계산식

### 출석률

| 항목 | 내용 |
|------|------|
| **표준 표시명** | `출석률` / `평균 출석률` |
| **공식** | (전체 − NOT_ATTEND − 패널티//3) / (전체 − 중도탈락미출석) × 100 |
| **NOT_ATTEND_STATUSES** | `{'결석', '중도탈락미출석', '100분의50미만출석'}` |
| **패널티** | 지각 + 조퇴 + 외출 누적 3건 → 가상 결석 1일 차감 |
| **참조 함수** | `utils.calc_attendance_rate()`, `utils.calc_attendance_rate_from_counts()` |

**예외**: `현재_운영_현황.py` 실시간 출석률은 `입실중` 상태 추가 포함.

---

### 수료율

| 항목 | 내용 |
|------|------|
| **표준 표시명** | `수료율` / `평균 수료율` |
| **KPI 공식** | FINI_CNT / TOT_PAR_MKS × 100 |
| **상세 공식** | `TRNEE_STATUS.str.contains('수료\|조기취업').sum()` / 총원 × 100 |
| **참조** | `home.py` (KPI 요약), `종료과정_성과.py` (개별 기수 재집계) |

> `== '수료'` 정확일치 금지. HRD-Net 값은 `'정상수료'`, `'80%이상수료'` 등.

---

### 취업률

| 항목 | 내용 |
|------|------|
| **표준 표시명** | `취업률 (3개월)` / `취업률 (6개월)` |
| **3개월** | EI_EMPL_RATE_3 |
| **6개월 합산** | EI_EMPL_RATE_6 + HRD_EMPL_RATE_6 |
| **특수코드** | A=개설예정, B=진행중, C=미실시, D=수료자없음 → `pd.NA` |
| **참조 함수** | `utils.calc_employment_rate_6()`, `utils.parse_empl_rate()` |

---

### 모집률

| 항목 | 내용 |
|------|------|
| **표준 표시명** | `모집률` / `평균 모집률` |
| **공식** | 신청인원 / 정원 × 100 |
| **상한** | 100% (초과분 clip) |
| **정원 0** | NA 처리 (분모 0 방지) |
| **월별 평균** | 신청인원 0명 과정 제외 |
| **참조 함수** | `utils.calc_recruit_rate()` |

---

### 만족도

| 항목 | 내용 |
|------|------|
| **표준 표시명** | `만족도 (100점 환산)` / `만족도(점)` (테이블) |
| **공식** | STDG_SCOR ÷ 100 (원본 10,000점 만점 → 100점 환산) |
| **가중 평균** | Σ(STDG_SCOR × 수강인원) / Σ(수강인원) ÷ 100 |
| **과정 평균** | 개별 과정 만족도의 단순 평균 |
| **참조** | `시장_분석.py` |

---

### 매출

| 항목 | 내용 |
|------|------|
| **표준 표시명** | `기준 매출` / `실제 매출` / `달성률` / `비례 손실액` |
| **일 훈련비** | 145,200원 (`config.DAILY_TRAINING_FEE`) |
| **기준 매출** | 훈련일수 × 일훈련비 × 수강생수 |
| **전액 청구** | 출석률 ≥ 80% → 기준 매출 전액 |
| **비례 청구** | 0% < 출석률 < 80% → 출석률 비례 |
| **미청구** | 출석률 = 0% |
| **달성률** | 실제 매출 / 기준 매출 × 100 |
| **비례 손실액** | 기준 매출 − 실제 매출 |
| **참조 함수** | `utils.calc_revenue()`, `utils.get_billing_periods()` |
