## 데드코드 방지

Python 파일을 편집할 때 다음을 준수한다:

1. **import 정리**: 새 import 추가 시 기존 미사용 import가 없는지 확인. 기능 제거 시 해당 import도 함께 삭제
2. **변수/상수 정리**: dict, list, 상수 등 정의만 있고 참조 0건인 코드는 남기지 않음
3. **잔류 코멘트 정리**: 제거된 기능에 대한 주석(`# ... → 제거`, `# TODO: 삭제 예정` 등)은 함께 삭제
4. **중복 import 금지**: 같은 모듈의 import가 파일 내 여러 곳에 있으면 하나만 남김 (파일 상단 유지)
