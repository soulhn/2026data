## UI 용어 표기 규칙

지표 표시명은 `GLOSSARY.md` 기준을 따른다.

### 띄어쓰기

- **수식어 + 지표**: 공백 필수 → `평균 모집률`, `평균 출석률`, `누적 총 매출`
- **복합명사**: 공백 필수 → `기준 매출`, `실제 매출`, `비례 손실액`, `중도 탈락`
- `st.metric` 레이블, 테이블 컬럼명, 차트 범례 모두 적용

### 단위 표기

- **st.metric**: 레이블에 단위 생략, 값에 포함 → label=`"평균 출석률"` value=`f"{rate:.1f}%"`
- **테이블 컬럼**: 괄호 단위 허용 → `수료율(%)`, `만족도(점)`
- **차트 축**: 공백+괄호 → `모집률 (%)`, `매출액 (원)`

### 기간 표기

- 취업률 기간: `취업률 (3개월)`, `취업률 (6개월)` — 공백+괄호 형식
- 금지: `취업률(3개월)`, `3개월취업률`, `3개월 취업률`
